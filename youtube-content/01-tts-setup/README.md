# TTS Setup Video - Production Folder

This folder contains all assets for the Text-to-Speech setup video.

## Folder Structure

- **slides/**: Slide images (PNG, 1920x1080)
- **audio/**: TTS narration files (MP3)
- **recordings/**: Screen recordings (MP4)
- **final/**: Final exported video (MP4)

## Naming Convention

- Slides: `slide-01.png`, `slide-02.png`, etc.
- Audio: `narration-01.mp3`, `narration-02.mp3`, etc.
- Recordings: `demo-recording.mp3`
- Final: `final-video.mp4`
