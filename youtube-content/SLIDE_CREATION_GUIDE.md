## ğŸ“ Step-by-Step Process

### Phase 1: Setup & Template (30 minutes)

1. **Choose your tool** (Canva recommended)
2. **Set dimensions**: 1920x1080 pixels (Full HD)
3. **Create template slide** with:
   - Your portfolio color scheme (purple/indigo)
   - Logo or branding element
   - Consistent font (sans-serif, readable)
   - Title area
   - Content area
4. **Save as template** for reuse

### Phase 2: Create Slides (2-3 hours)

For each of the 13 slides in `01-tts-setup-slides.md`:

1. **Read the slide script**:
   - Note the "Slide Text" content
   - Note the "Visual" description
   - Note the "TTS Narration" (for later)

2. **Create the slide**:
   - Use your template
   - Add the slide text (keep it minimal - 3-5 bullet points max)
   - Add visual elements (icons, diagrams, screenshots)
   - Ensure text is readable (at least 24pt font)

3. **Design principles**:
   - **Less is more**: Don't overcrowd
   - **Visual hierarchy**: Title larger, content smaller
   - **Consistent spacing**: Same margins/padding
   - **Color contrast**: Text readable on background
   - **Dark mode friendly**: Test on dark background

4. **Export**:
   - Save as: `slide-01.png`, `slide-02.png`, etc.
   - Resolution: 1920x1080
   - Format: PNG (best quality)

### Phase 3: Special Slides

#### Title Slide (Slide 1)
- Make it impressive
- Include portfolio preview/screenshot
- Large, bold title
- Subtitle with key tech

#### Architecture Diagram (Slide 3)
- Use arrows or flow diagram
- Simple boxes with labels
- Clear flow direction
- Can use Canva's diagram tools

#### Code Snippets (Slides 4-7)
- Use code-style font (monospace)
- Syntax highlighting if possible
- Keep snippets short (3-5 lines)
- Add comments/annotations

#### Demo Slide (Slide 9)
- Can be a placeholder
- Or screenshot of the feature
- Text: "See it in action!"

## ğŸ¨ Design Tips

### Color Scheme
Use your portfolio colors:
- **Primary**: Purple (#9333EA or similar)
- **Secondary**: Indigo (#4F46E5 or similar)
- **Background**: White/Light gray (or dark for dark mode)
- **Text**: Dark gray/Black (or white for dark mode)
- **Accents**: Use purple for highlights

### Typography
- **Title**: Bold, 48-60pt
- **Subtitle**: Medium, 32-40pt
- **Body**: Regular, 24-32pt
- **Code**: Monospace, 20-24pt
- **Font**: Sans-serif (Arial, Helvetica, Inter, etc.)

### Layout
- **Rule of thirds**: Don't center everything
- **White space**: Leave breathing room
- **Alignment**: Keep things aligned
- **Consistency**: Same layout pattern

### Visual Elements
- **Icons**: Use simple, consistent icons
- **Screenshots**: Add subtle border/shadow
- **Diagrams**: Keep them simple
- **Bullet points**: Use checkmarks or arrows
